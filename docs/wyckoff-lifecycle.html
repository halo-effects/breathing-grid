Six-phase market cycle engine — accumulate, ride, distribute, short, spring, repeat.

The Wyckoff Cycle — What AIT Does at Every Phase

Idealized price action with AIT strategy overlay

DCA — Accumulation

Grid trading in ranging/accumulation markets. Small take-profit captures on each grid level. Default starting phase.

→ EXIT: TA Score ≥50 + F&G ≥80 + ATH gate (no weekly veto)
→ MARKUP: Regime TRENDING + price > 50d high + ADX >25 + volume (weekly confirmed)

MARKUP — Ride Uptrend

Large position riding the uptrend. Deploy 50/70/90% of capital (by profile). Hold until distribution signals fire.

→ EXIT: TA Score ≥50 fires during markup → close position, begin lot unwind

EXIT — Lot Unwind

Methodical position unwind. Rally selling, trailing stops, time-based urgency escalation. Used after both DCA top signals and MARKUP distribution.

→ MARKDOWN: All lots fully unwound → deploy shorts

MARKDOWN — Short Decline

Aggressive shorts riding the decline. Tiered 50/70/90% deployment by profile. Capture downside momentum.

→ SPRING: Price drops 25%+ from exit price → close shorts, deploy longs
→ DCA: Price recovers above entry (false exit) → return to grid

SPRING — Buy the Bottom

Tiered long deployment at 25/28/35% discount from exit price. Catch the capitulation bounce.

→ MARKUP: Price recovers 25%+ from spring entry + regime TRENDING
→ DCA: 60-day timeout with no recovery signal

The Full Cycle

DCA accumulates → breakout triggers MARKUP → distribution signals EXIT → lots unwind into MARKDOWN → price drops trigger SPRING → recovery returns to MARKUP or timeout falls back to DCA.

Every bear market ends. Every bull market ends. The engine adapts.

Signal Components

TA Scorer (100pts max)

  • RSI bearish divergence — 25pts
  • Volume divergence — 25pts
  • Wick rejection (long upper) — 25pts
  • Momentum stall (ROC fade) — 25pts

Pi Cycle Indicator

  • 111-SMA crosses 2×350-SMA — 25pts
  • Gap < 2% (near cross) — 15pts
  • Gap < 5% (approaching) — 5pts

Fear & Greed Index

  • F&G ≥ 90 (extreme greed) — +25pts
  • F&G ≥ 80 (greed) — +15pts
  • F&G ≤ 20 (fear) — -10pts
  • F&G ≤ 10 (extreme fear) — -20pts

Gates & Thresholds

  • Exit threshold: 50 points
  • ATH gate: price within 25% of ATH
  • Weekly veto: markup needs weekly confirm
  • No weekly veto on exit signals
